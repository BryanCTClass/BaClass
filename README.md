# Intro to BA & Salesforce

A hands-on course covering Business Analysis fundamentals and Salesforce development using a real developer org.

## Course Scenario

We follow **SoundCraft Speakers**, a fictional manufacturer of high-end audio equipment. All exercises, user stories, and data are built around their sales and operations team.
